Bevor es los gehen kann:

- Node JS muss installiert sein: https://nodejs.org/





Setup des react.js-Projekts:

1. Konsole im Admin-Modus öffnen und in Root-Verzeichnis des Projekts wechseln (=> react-skeleton)

2. "npm run setup" ausführen



Build des Projekts durchführen:

1. Konsole im Admin-Modus öffnen und in Root-Verzeichnis des Projekts wechseln (=> react-skeleton)

2. "npm run build" ausführen



Die Anwendung ausführen:

1. Konsole im Admin-Modus öffnen und in Root-Verzeichnis des Projekts wechseln (=> react-skeleton)

2. "npm run serve" ausführen

3. Im Browser die URL "localhost:8080" aufrufen




Ordnerstruktur:

bower_components: Enthält alle installierten Bower-Module

node_modules:     Enthält alle installierten NodeJS-Module

dist:             Enthält die kompilierte Version der Anwendung (Diese Dateien werden auf dem Zielsystem abgelegt)

pre:              Enthält die kompilierten Javascript-Dateien

src:              Enthält die Quelldateien der Anwendung für die Entwicklung (z.B. React TSX-Komponenten)
